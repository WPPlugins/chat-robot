This plugin add in the footer of body a new tag script that will activate your ChatRobot.

== Description ==

Chat Robot installs your 'Chat Robot' to your blog.

To activate ChatRobot, you need to enter the TOKEN code of your source, then check the box to activate the plugin.
This will add in the footer of body a new tag script that will activate your ChatRobot.

Technical support:
If you need help with the installation or use of the script, contact us info@chat-robot.com or entering our website http://www.chat-robot.com.

== Installation ==

1. Upload `chat-robot` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (go to the Settings/Chat Robot)
